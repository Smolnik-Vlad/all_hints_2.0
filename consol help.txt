cls - очистить консоль
cd Имя - перейти в папку 
D: - перейти на диск
mkdir имя - создать папку
rmdir/S /Q D:\work\loh2 - удалить папку loh2 (/S - удалить все внутренности; /Q - удалить без запросов)
del имя - удалить файл (нужно зайти в папку с файлом)
NUL>ИМЯ - создать файл
echo text>>file.txt - вписать текст в файл

move имя1 имя2 - переименовывает папку
move имя_файла имя_папки - перемещает файл в указанную папку
move D:\catalog1\text1 D:\catalog2\text2 - перемещает файл из однй папки в лругую, переименовывая ее
