	Teги блочные и строчные:
<h1></h1> (1 до 6) - теги для заголовков (блочный) 
<p></p> - текстовый абзац (обычны текст)(блочный)


<span></span> - тег, растягивающийся на длину текста (после не отделяется от других cтрочных тегов)(строчный)
<a target="" href=""></a> - тег-ссылка(строчный тег)
<img src="" alt=""/> - тег-картинка(пишется без закрыв элемента), обязательные атрибуты: src - расположение картинки, alt - альтернативный текст

	Тег <form> и все его прилегающие
<form action="#" method="" enctype = ""></form> - тег, отвечающий за передачу данных(блочный)
<input type="" name="" value=""> - тег ввода (далее все смотреть в атрибутах) 
<textarea name="" cols="" rows=""> - тег для объемного ввода
<button type=""></button> - кнопка

	Атрибуты:
Для любых тегов:
	disabled - делает все теги не октивированными, тусклыми(нельзя вводить инфу) (все, кроме form)
	tabindex="1" - порядок перехода между элементами при нажатии tab
	class="" - класс тега (используется при обрапботке в scc)

<a>:
	href=""  - атрибут с ссылкой; 
	target="" - атрибут со свйствами: _blank (открытие  новом окне браузер)

<form>:
	action="" - куда полетя данные(# - заглушка)
	method="" - метод передачи данных: get - идет прямо в аресной строке (малая передача данных)
					   post - передает скрытно, используя запрос браузера (большая передача данных)
	enctype = "multipart/from-data" - вместе с атрибутом method="post" позволяет отправлять файлы

<input>:
	type="" - метод приема данных от пользователя:  text - input примит обличие поля воода
						        tel - для ввода цифр (переключит клавиатуру телефона на цифровую)
						        password -  вводимый текст шифруется звездочками
						        radio - создается радио кнопка (чтобы их связать, имя должно быть одинаково)
						        checkbox - создаются несвязанные флажки (ИМЯ РАЗНОЕ!!!)
							file - создает кнопку для добавления файла (value не используется)
							 
							
	name="" - имя объекта, нужно для соединения объектов НЕ ДОЛЖНЫ ПОВТОРТЬСЯ (искл: type="radio") ( в type = "text" name ="username" - предназначена для ввода логина (дает подсказки логинов)
			    			     		 	    						 name ="userpass" - для пароля
															 name = "name" - выводит имена)
	value="" - значение (при type="text" значение value пишется в окошке)
	
	checked - атрибут для установления галочки при type="radio" (указывается на той кнопке, которую хотим указать)
	
	placeholder="" - для написания водяного текста в поле (только для type="text")
	
	

<textarea>:
	name="" -  имя объекта
	cols="40" - длина рамки (обычно убирается)
	rows="10" - ширина рамки (обычно убирается)
	placeholder="" - для написания водяного текста в поле 
	
<select>:
	multiple - для множественного выбора
<option>:
	selected - установление по умолчанию составляющего списка

<button>:
	type="" - атрибут обработки:
					submit - отправляются все заполненные поля
					reset - сброс настроек

	Мета-теги(пишутся в head)
<meta name="" content=""> - мета тег

<meta charset="UTF-8"> - мета-тег для настройки языка
<meta name="viewport" content="width=1170"> - мета-тег для фиксации верстки, то есть условие адаптации при изменении размера экрана (name="viewport" - способ отображения, content="width=1170" - заданый размер)
		      content ="width=device-width" - если мы хотим, чтобы изменялось
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=0/1"> - мета-тег для настройки масштабирования на телефоне. (initial-scale - изначальный масштаб страниицы, maximum-scale - максимальный масштаб страницы, user-scalable = 0/1 - запрещаем/разрешаем масштабирование страницы)
 <meta name="format-detection" content="telephone=no"> - для отключения возможности нажать на телефон

сео-теги
<meta name="description" content="что такое ..."> - краткое описание страницы
<meta name="keywords" content="мета теги, meta, мета теги для сео"> - ключевые слова (не более 20)
<meta name="robots" content="noimageindex, nofollow">доступ поисковых роботов к странице (в данном случае запрещает проходить индексацию по картинкам), всю инфу о индексации смотреть в инете

технические теги:
<meta name="Author" content="Vlad"> - мета-тег автора
<meta name="Copyright" content="Vlad"> - владелец авторских прав
<meta name="Adress" content=""> - адрес
<meta http-equiv="refresh" content="0, url="> - мета-тег перенаправляет на другую страницу, content="0, url=" - количество секунд, через сколько нужно перенаправить, url= - адресс
